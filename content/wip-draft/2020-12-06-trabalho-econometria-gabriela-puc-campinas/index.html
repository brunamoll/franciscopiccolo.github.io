Regresso Linear Mltipla com Dados do Mercado de Trabalho

Introduo

Neste post vou desenvolver um exerccio comumente passado em aulas de econometria, envolvendo regresso linear mltipla. O dataset contm dados do mercado de trabalho, com os seguintes campos:

  • Ano
  • Trimestre
  • UF
  • Situao da casa (house_situation)
  • Gnero: 1 = Homem; 2 = Mulher
  • Idade
  • Raa: 1 = Branco; 2 = Preta; 3 = Amarela; 4 = Parda; 5 = Indgena
  • Tipo de emprego
  • Principal atividade
  • Anos de estudo
  • Rendimento mensal

Para realizar este exerccio, irei utilizar os seguintes pacotes:

O modelo ser uma regresso linear mltipla tendo monthly_earning como varivel dependente. O modelo poder ser descrito de acordo com a equao abaixo:

\[\hat{Y} = \hat{\beta_1}X_1 + \hat{\beta_2}X_2 + \hat{\beta_3}X_3 + \hat{\beta_4}X_4 + \mu \]

Vamos ver separadamente qual a correlao da varivel dependente com cada uma das variveis independentes, para poder antever os resultados do modelo.

Modelo Final

No modelo final, ser retirado os valores zerados de renda mensal.

# Criando varivel com resultado do modelo de regresso mltipla
dataset_1 %>% 
  filter(monthly_earning > 0) -> dataset_1

dataset_1 %>% 
  mutate(dummy_race = case_when(race == "branco" ~ 1,
                                TRUE ~ 0)) %>% 
  lm(formula = monthly_earning ~ (years_of_study + dummy_race)) -> final_model
final_model %>% 
  summary() %>%
