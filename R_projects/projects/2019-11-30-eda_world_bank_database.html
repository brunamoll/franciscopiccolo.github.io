Neste post irei analisar alguns indicadores do Banco Mundial, que sero obtidos facilmente atravs de uma API desta instituio (World Bank Data API). Existe um pacote criado no R que pesquisa e extrai dados desta API, o pacote o wbstats criado em Janeiro de 2018 por Jesse Piburn.

O Banco Mundial foi criado em 1944 nas conferncias de Bretton Woods e possui como objetivo combater a pobreza e a desigualdade, atravs de investimentos e conselhos para pases em desenvolvimento. A misso encontrada em sua pgina inicial de reduzir para 3% a parcela da populao vivendo na extrema pobreza at 2030, porm uma de suas primeiras atribuies foi ajudar a reconstruir a infraestrutura do continente europeu que estava destruda pela segunda guerra mundial 1.

Para execuo de seu propsito de maneira eficiente, se faz necessrio estudos que demandam um grande volume de dados para embasar suas decises de investimento. Este grande volume de dados foi, e continua sendo construdo pela instituio, podendo ser encontrado nesta pgina.

O pacote wbstats trabalha de maneira bem simples, onde o principal desafio em sua utilizao identificar qual dado voc quer extrair, pois tem-se basicamente um data lake disponvel com informaes que voc nem imagina que tenham sido coletadas.

Neste demonstrativo, irei analisar alguns indicadores econmicos como PIB, dvida e ndice Gini, visando mostrar como fcil extrair e analisar os dados com o uso do wbstats. Alm do pacote wbstats, irei usar outros pacotes que esto descritos abaixo:

A primeira funo do pacote a wb, que faz o dowload dos dados solicitados na API e retorna um data frame. Os principais parmetros da funo so: indicator, country, startdate e enddate. O indicator um cdigo que pode ser observado na URL da pgina do Banco Mundial, por exemplo, para o indicador do PIB per capital a valores do dlar de 2010, a URL (https://data.worldbank.org/indicator/NY.GDP.MKTP.KD?view=chart) e o cdigo do indicador para inserir na funo o NY.GDP.MKTP.KD. O parmetro country pode ser um pas no formato iso2c (e.g.BR,AF ou US), uma regio (e.g.1A que representa os pases rabes), todos os pases e regies usando all no parmetros ou apenas pases usando countries_only. O startdate e enddate so as datas de interesse sobre o indicador em questo, onde estes campos podem ser substituidos pelo parmetro mrv (most recent value) que receber um valor inteiro (e.g.4) e ir trazer os 4 valores mais recente do indicador.

Para mostrar um pouco do processo de extrao dos dados desta API e analisar como esto organizados os indicadores desta base de dados, vamos analisar alguns indicadores. Neste post, irei analisar especificamente os indicadores que esto no grupo Economy & Growth.

Fiz uma classificao simples do PIB per Capita para conseguir segmentar a anlise por tipo de pas (e.g.low income, middle, high).

df_gdp_pc_2 <- wbstats::wb(indicator = "NY.GDP.PCAP.CD",
                           country = "countries_only",
                           startdate = 1990,
                           enddate = 2010)

df_gdp_pc_2 %>%
  select(
    date,
    country,
    value
  ) %>% 
  filter(date %in% c("1990",max(date))) %>% 
  mutate(
    gdp_90 = ifelse(date == "1990",value,0),
    gdp_10 = ifelse(date == "2010",value,0)
  ) %>% 
  group_by(country) %>% 
  summarise(
    gdp_90 = sum(gdp_90),
    gdp_10 = sum(gdp_10)
  ) %>% 
  filter(gdp_90 > 0 & gdp_10 > 0) %>% 
  mutate(
    delta = round(1-(gdp_90/gdp_10),digits = 4),
    income_range = ifelse(gdp_10 <= 5000,"low",
               ifelse(gdp_10 <= 15000,"middle",
               ifelse(gdp_10 <= 45000,"up_middle",
               ifelse(gdp_10 <= 60000,"high",
               ifelse(gdp_10 <= 90000,"up_high",
                      "ultra_rich")))))
  ) %>% 
  filter(income_range == "low") %>% 
  arrange(desc(delta)) %>% 
  head(20) %>% 
  ggplot()+
  ggalt::geom_dumbbell(aes(y = fct_reorder(country,gdp_10),
                           x = gdp_90,
                           xend = gdp_10),
  colour = "#3E39CD", size = 1,
  colour_x = "#BD274A", colour_xend = "#60AF1A")+
  scale_x_continuous(labels = scales::comma)+ # separando por vrgula ('000)
  theme(axis.title.x = element_blank(),
        axis.title.y = element_blank())+
  labs(
    title = "Os 20 pases com maior variao de PIB per capita",
    subtitle = "Categoria: Low income (GDP per capital <= 5.000 US$)"
  )

O cdigo anterior pode ser replicado, alterando apenas os parmetros filter(income_range ==low) para middle, up_middle, high e up_high, ou at memso retirando este filtro e trazendo todos os pases.
